# Logiko – Logistics & Transport Platform

A modern logistics tracking web app built with React, TypeScript, Vite, and Tailwind CSS.

## Features
- 📦 Real-time shipment tracking with animated maps
- 📊 Dashboard with charts, status donuts, and live stats
- 🚚 Multi-modal freight services (Sea, Air, Road, Rail)
- 🔔 Notifications, invoices, addresses, and support pages
- 📱 Fully responsive with collapsible mobile sidebar

## Stack
- **Frontend**: React 18 + TypeScript
- **Bundler**: Vite 5
- **Styling**: Tailwind CSS 3
- **Icons**: Lucide React

## Getting Started

```bash
npm install
npm run dev
```
